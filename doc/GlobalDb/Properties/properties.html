Свойства

Для просмотра свойств и их значений будет использоваться окно с названием "Свойства".

1.Добавление свойств

Для того, чтобы добавить свойство, нужно выбрать нужный материал, а затем его модель. После чего нажать на знак плюса в правом верхнем углу.

Затем в появившемся окне написать название свойства, его описание и нажать "Ок".

2.Добавление значений свойств

Для того, чтобы добавить значение свойства, нужно выбрать нужный материал, нажать на правую кнопку мыши и затем выбрать "Задать значение свойству материала".

Затем в появившемся окне выбрать нужное свойство, написать его значение и нажать Задать значение".

Добавление значения свойства материала возможно только при сопоставлении этого материала с моделью этого свойства!

3.Удаление свойств

Чтобы удалить нужное свойство, необходимо навестись на него курсором, нажать правую кнопку мыши и затем выбрать "Удалить свойство из глобальной БД".
