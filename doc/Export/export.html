Перенос данных из локальной базы данных в глобальную

Для того, чтобы перенести данные из из глобальной базы данных в локальную, нужно выбрать материал в локальной базе данных.

Затем выбрать нужную конечную ветку(не содержащей в себе других веток) в окне "Классификации".

После этого нужно нажать на кнопку "Экспорт".
